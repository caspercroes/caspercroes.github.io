A classic 90s style horror-themed action-adventure game set in a 1920s-inspired fantasy universe. You play as an Elite Royal Agent named Alisa who, while chasing a wanted criminal, ends up in an old victorian mansion. You'll have to find a way out while being haunted by mechanized creatures. Can you survive the Dollhouse?
